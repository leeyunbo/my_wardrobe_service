## 나만의 옷장 서비스 개발 

```text
옷들을 구매하고 옷장에 넣어 놓은 후 까먹는 상황이 비일비재하다. 
그런 사람들을 위해 자신들의 옷장을 관리하고 더 나아가 다른사람들과 옷장을 공유하여 
옷과 관련된 정보를 주고 받을 수 있는 커뮤니케이션 사이트
```
